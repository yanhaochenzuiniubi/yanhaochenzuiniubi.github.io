如何安装jdk

JDK的安装方法及其配置

JDK下载地址https://www.oracle.com/technetwork/java/javase/downloads/index.html

输入变量名:JAVA_HOME

变量值：C:\Program Files\Java\jdk-12.0.2 （JDK的存放路径）

配置Path

找到系统变量中的Path 双击打开

输入：%JAVA_HOME%\bin

输入变量名：CLASSPATH

变量值：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar

Win+R组合键输入cmd 打开CDM

输入命令java -version 检查版本号

输入命令javac 检查Path配置情况

JDK JRE JVM的区别和关系

JDK

Java Development Kit (Java开发者工具包)

内容有Java、javac、jar、Javadoc等。并且包括JRE和JVM

JRE

Java Runtime Environment (Java运行环境)

内容有AWT,Beans,JDBC,Swing（JavaSE API）等。

从这里可以看出JDK包含JRE，父子关系，在JRE之上扩充了一些开发工具，在实际应用下载的时候就简简单单下载JDK就好了

JVM

Java Virtual Machine(Java虚拟机)

可以把他当成一种规范，用软件也可用硬件实现，用电脑模拟一个CPU去处理Java相关的东西，Java的跨平台核心就是JVM，可以在不同的操作系统上运行

JDK JRE JVM的区别和关系

JDK

Java Development Kit (Java开发者工具包)

内容有Java、javac、jar、Javadoc等。并且包括JRE和JVM

JRE

Java Runtime Environment (Java运行环境)

内容有AWT,Beans,JDBC,Swing（JavaSE API）等。

从这里可以看出JDK包含JRE，父子关系，在JRE之上扩充了一些开发工具，在实际应用下载的时候就简简单单下载JDK就好了

JVM

Java Virtual Machine(Java虚拟机)

可以把他当成一种规范，用软件也可用硬件实现，用电脑模拟一个CPU去处理Java相关的东西，Java的跨平台核心就是JVM，可以在不同的操作系统上运行

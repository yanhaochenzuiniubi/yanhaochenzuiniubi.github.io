今天教大家如何关掉服务器的IE浏览器的增强提示

Windows Server 2012R关闭IE增强安全模式方法

最近买了一台服务器，在 Windows Sever 2012 中打开 IE 浏览器时，IE10 会出现【已启用 Internet Explorer 增强的安全配置】的提示信息，就很TMD烦，打开新业标签也是一样的。

【问题截图忘记了.jpg】不好意思哈哈哈哈

问题的原理

原理是因为服务器不是用来上网用的，考虑到安全的问题哈哈，可能会增强被攻击的疑虑！

解决办法

1.首先先打开开始，那个windows小图标，找到服务器管理器，并打开

![](D:\yanhaochenzuiniubi blog\yanhaochen\source_posts\hello-word3\捕获2.PNG)

2.点击本地服务器，有一个IE增强的安全配置，点击那个启动（蓝色那个）我这里已经关闭了。

![](D:\yanhaochenzuiniubi blog\yanhaochen\source_posts\hello-word3\捕获3.PNG)

![](D:\yanhaochenzuiniubi blog\yanhaochen\source_posts\hello-word3\捕获4.PNG)

都关掉，关闭，点确定。

然后重启浏览器就好了，现在问题又来了，现在又提示Internet Explorer增强的安全配置未启用。

![](D:\yanhaochenzuiniubi blog\yanhaochen\source_posts\hello-word3\捕获.PNG)

很简单，设置重新启动页面就得

![](D:\yanhaochenzuiniubi blog\yanhaochen\source_posts\hello-word3\捕获5.PNG)

重启就好了，100%可靠哦。啦啦啦和❤😊
